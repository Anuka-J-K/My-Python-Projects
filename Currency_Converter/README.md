# 🌍 Currency Converter in Python 🐍

A simple Python application that fetches live exchange rates from an API and converts currencies (e.g., USD to LKR). 💵

## ✨ Features
- **🌐 Live API Integration:** Fetches real-time exchange rates using the Exchangerate-API.
- **✅ JSON Parsing:** Extracts data from API responses efficiently.
- **⚠️ Error Handling:** Uses `try-except` blocks to handle network errors and connection issues gracefully.
- **🇱🇰 Sinhala Support:** Terminal encoding configuration to display Sinhala text and emojis properly.

## 🚀 How to Use
1. Make sure you have [Python](https://python.org) installed. 🐍
2. Install the `requests` library if you haven't already:
   ```bash
   pip install requests
   ```
3. Open your terminal or command prompt. 🖥️
4. Run the script using:
   ```bash
   python main.py
   ```

## 📂 Files in this Project
- `main.py`: The main program file containing the API request and parsing logic. 📝
