# 🎂 Age Calculator in Python 🐍

A simple and interactive Python application that calculates a user's age based on their birth year and a target future year. 📅

## ✨ Features
- **📥 User Input:** Takes birth year and target year from the user.
- **✅ Validation:** Checks if years are negative or if the birth year is greater than the future year.
- **⚠️ Error Handling:** Uses `try-except` blocks to handle non-numeric inputs gracefully.

## 🚀 How to Use
1. Make sure you have [Python](https://python.org) installed. 🐍
2. Clone or download this repository. 📥
3. Open your terminal or command prompt. 🖥️
4. Run the script using:
   ```bash
   python "Age calculation program1.py"
   ```

## 📂 Files in this Project
- `Age calculation program.py`: The basic version of the logic. 📝
- `Age calculation program1.py`: An improved version with error handling and validation. ⭐
